## Getting Started

### Prerequisites

* node.js
* npm

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/lopezrunco/Agroagua-web.git
   ```
2. Install NPM packages
   ```sh
   npm i
   ```
3. Start app
   ```sh
   npm start
   ```
3. Build app
   ```sh
   npm run build
   ```
